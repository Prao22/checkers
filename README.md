# Chinese Checkers
In pairs create application for playing chinese checkers
#### Project for Software Systems Engineering Course
For game rules see [wikiHow.com](https://www.wikihow.com/Play-Chinese-Checkers).

### Features
- Application operates on the client - server architecture.
- The server verifies whether the movements are correct.
- Played games and moves are saved into database.
- Client is able to watch replays of played games.
- Parameters, rules are easy to change.
